# 🚀 Any-Router: 极简个人意图路由与自动化 Agent

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Python](https://img.shields.io/badge/python-3.8%+-blue)
![Model](https://img.shields.io/badge/model-DeepSeek--V4%20%7C%20MiMo--V2.5-orange)

**Any-Router** 是一个极致轻量、超低成本的后台静默自动化 Agent。它完全抛弃了传统的“多轮对话”模式，专注于**“单步意图解析”**。通过将用户的碎片化自然语言输入转化为结构化的 JSON 指令，Any-Router 能够无缝触发外部系统的 API（如日历、记账本、知识库、智能家居），成为你数字生活的超级中枢。

---

## ✨ 核心特性

- ⚡️ **无状态与超低 Token 消耗**：采用 Stateless 架构，无需携带历史上下文。结合极简 System Prompt，单次请求总消耗控制在 200 Tokens 以内。每月仅需十余元人民币预算（基于 DeepSeek-V4）即可支撑数万次自动化调度。
- 🧩 **100% 纯 JSON 输出**：强制开启大模型 `json_object` 模式，确保输出高度格式化，零幻觉干扰，直接对接下游代码逻辑。
- 🔌 **高扩展性路由**：只需在提示词中新增意图分类（Action），并在代码中添加对应的 API 调用函数，即可无限扩展 Agent 的能力边界。
- 🏠 **面向全模态与 IoT 演进**：架构设计预留多模态接口，计划深度接入小米 MiMo 大模型生态与米家（Mi Home）智能设备。
